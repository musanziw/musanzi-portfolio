## Portfolio

### Personnal website

## Requirements
 * composer : 2.3.10
 * php : 8^
 * npm : 8^ or yarn : 1.23^
 * laravel : 9^

## Install dependencies

`$ composer install 
`
<br />
`
$ yarn install or npm install
`

## Lunch the project 

`
$ yarn dev or npm run dev
`
<br />
`
$ php artisan serve
`
